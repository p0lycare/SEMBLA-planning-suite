Projekt / Export

Die obigen DXF/IFC laufen sofort im Browser. Für normnahe, CAD-geprüfte Dateien das Projekt-JSON exportieren und im Suite-Ordner ausführen:
npm run dxf -- projekt.json plan.dxf · npm run dxf -- projekt.json ansichten.dxf --ansichten (ezdxf)
npm run ifc -- projekt.json modell.ifc (IfcOpenShell) · npm run validate:ifc -- modell.ifc (web-ifc-Prüfung)
